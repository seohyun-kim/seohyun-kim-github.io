## 메인 페이지 UI  

(이미지 첨부)


#### ✔ Done  
```
1. 계획했던 레이아웃과 유사하게 배치 
   ( 속도, 건물 출입 등등 아직 판별 불가한 데이터는 그냥 위치만 잡음 )
    
2. 일간/ 기간 버튼 탭 설정 ( 페이지 변화, 탭 색상 변화 )

3. 해당 날짜의 데이터 텍스트 출력, 그래프, 파이차트 출력
   ( 현재는 전체 데이터로 불러오게 됨. 추후에 1시간 단위로 계산된 테이블로부터 계산되도록 수정 )
```
